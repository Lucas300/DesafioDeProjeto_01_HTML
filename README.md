Neste projeto, foram abordados todos os recursos de HTML aprendidos até o momento no curso. Utilizando a estrutura básica do HTML, foram explorados os seguintes elementos:

- Títulos e subtítulos para organizar o conteúdo da página de forma hierárquica e significativa.
- Parágrafos para separar e estruturar o texto em unidades de significado.
- Elementos de formatação, como negrito, itálico, sublinhado, entre outros, para destacar e enfatizar partes específicas do conteúdo.
- Importação de imagens para adicionar elementos visuais à página, tornando-a mais atrativa e informativa.
- Tipos de listas, incluindo listas ordenadas e não ordenadas, para organizar informações de maneira estruturada e legível.
- Tipos de links para fornecer navegação e acesso a recursos externos, como páginas da web, documentos e mídias.
- Referências do projeto, que detalham cada elemento utilizado no projeto, explicando sua função e importância na construção da página.
Cada item foi explicado detalhadamente, demonstrando como aplicá-lo no desenvolvimento de uma página HTML.
