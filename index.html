Estrutura básica HTML

(voltar)

HTML, ou HyperText Markup Language, é a linguagem fundamental para a criação de páginas web. Sua estrutura básica consiste em elementos que organizam e formatam o conteúdo para ser exibido em navegadores da web

Exemplo de Código:

        <!DOCTYPE html>
        <html lang="pt-br">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Exibindo Código HTML</title>
        </head>
        <body>
            <h1>Olá, Mundo!</h1>
        </body>
        </html>

Títulos e Subtítulos

(voltar)

Os títulos em HTML são elementos fundamentais para estruturar e hierarquizar o conteúdo de uma página da web. Eles são representados pelas tags <h1> até <h6>, onde <h1> é o título mais importante e <h6> é o menos importante.

Exemplo de Código:

        <h1> Título 1 </h1>
        <h2> Título 2 </h2>
        <h3> Título 3 </h3>
        <h4> Título 4 </h4>
        <h5> Título 5 </h5>
        <h6> Título 6 </h6>

Resultado:

Título 1

Título 2

Título 3

Título 4

Título 5
Título 6

Parágrafos

(voltar)

Os parágrafos em HTML são elementos utilizados para organizar e estruturar o conteúdo textual em uma página da web. Eles são representados pela tag <p>, que define um bloco de texto. Parágrafos são úteis para separar ideias e tornar o conteúdo mais legível para os usuários.

Exemplo de Código:

<p> Isto é um parágrafo </p>

Elementos de formatação

(voltar)

Os elementos de formatação do HTML são tags que permitem modificar a aparência do texto e do conteúdo em uma página da web como...fazer uma marcação no texto,colocar um sublinhado,colocar um trecho do texto em itálico entre outros aspectos que ela pode modificar no texto

Exemplo de Código:

<b> Texto em negrito </b>
<strong> Texto importante </strong>
<i> Texto em itálico </i>
<em> Texto enfatizado </em>
<mark> Texto marcado </mark>
<small> Texto Menor </small>
<del> Texto excluído </del>
<ins> Texto inserido </ins>
<sub> Texto subscrito </sub>
<sup> Texto sobrescrito 22 </sup>

Imagens

(voltar)

A importação de imagens em HTML é uma técnica fundamental para adicionar elementos visuais a uma página da web. Isso é feito por meio da tag <img> que permite incorporar imagens diretamente no conteúdo HTML.

Importando uma imagem do computador

Ao criar uma página da web, você pode importar imagens diretamente da sua própria máquina para adicionar elementos visuais personalizados ao conteúdo. Isso é feito utilizando a tag em conjunto com o atributo src, onde você especifica o caminho local para o arquivo de imagem no seu computador. Por exemplo:

Exemplo de Código:

< img width="200px" src="img/java.jpeg" alt="Imagem Java" >

Importando uma imagem da WEB

Além de importar imagens da sua própria máquina, você também pode utilizar imagens hospedadas na internet para adicionar recursos visuais à sua página da web.

Exemplo de Código:

< img width="200px" src="https://azilen-technologies.s3.us-west-1.amazonaws.com/wp-content/uploads/2023/07/spring.jpg" alt="Imagem SpringBoot" >

Listas

(voltar)

Listas em HTML são usadas para organizar e apresentar informações de forma estruturada e hierárquica em uma página da web. Existem dois tipos principais de listas em HTML: listas ordenadas e listas não ordenadas.

Listas Ordenada

Listas ordenadas são criadas usando a tag <ol> e também contêm itens de lista representados pela tag <li>. Essas listas são usadas quando a ordem dos itens é importante, como em uma lista de instruções passo a passo ou em uma lista de classificação.

Exemplo de Código:

       <ol>
            <li>Item 1</li>
            <li>Item 2</li>
            <li>Item 3</li>
                <ol>
                    <li>SubItem 3.1</li>
                    <li>Item 3.2</li>
                </ol>
        </ol>

Resultado:

  1. Item 1
  2. item 2
  3. Item 3
    1. SubItem 3.1
    2. SubItem 3.2

listas não Ordenada

As listas não ordenadas são criadas usando a tag <ul> e consistem em itens de lista representados pela tag <li>. Essas listas são usadas quando a ordem dos itens não é importante, como em uma lista de itens de navegação ou em uma lista de recursos.

Exemplo de Código:

       <ul>
            <li>Item 1</li>
            <li>Item 2</li>
            <li>Item 3</li>
                <ol>
                    <li>SubItem 3.1</li>
                    <li>Item 3.2</li>
                </ol>
        </ul>

Resultado:

  • Item 1
  • item 2
  • Item 3
    • SubItem 3.1
    • SubItem 3.2

Links

(voltar)

Links em HTML, representados pela tag <a>, desempenham um papel essencial na navegação na web, permitindo que os usuários naveguem entre diferentes páginas da web, sites e recursos online com facilidade.

Links Externos

Geralmente links que levam sites externos

DIO.ME

Exemplo de Código:

<a> href="https://www.dio.me/" target="_blank">DIO.ME</a>

Links de URL relativa

Links que navegam dentro da própria página ou dentro de um recurso no mesmo servidor

Voltar ao índice

Exemplo de Código:

<a> href="#indice">Voltar ao índice</a>
